<Steps>
  <Step title="Download Hanggent">
    Head to our official website to download the latest version.
    - Download for macOS
    - Download for Windows

```
<Warning>
  **macOS Prerequisite**
  Please ensure you are running macOS 11 (Big Sur) or a newer version to install Hanggent.
</Warning>
```

</Step>
  <Step title="Install the Application">
    - **On macOS:** Open the downloaded `.dmg` file and drag the Hanggent icon into your Applications folder.
    - **On Windows:** Run the downloaded `.exe` installer and follow the on-screen instructions.
  </Step>
</Steps>

Once installed, launch Hanggent and log in to get started!
